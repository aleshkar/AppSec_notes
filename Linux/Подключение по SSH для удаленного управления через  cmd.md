Итак, очень важная тема, каким образом вам подключиться допустим с винды к Linux виртуалке? Все просто:
1. Необходимо машину на Linux подключить к одной сети с Windows машиной, а именно(мостовой способ подключения выбрать в самом VM через настройки)
2. После этого нам надо узнать ip-addres сети Linux машины, командой hostname -I сделаем это и получим ip
3. Далее пишем в командной строке(cmd) на Windows: ssh hostname@ip-addres, всё, мы подключились и теперь с винды удаленно можем что-то выполнять в командной строке.
