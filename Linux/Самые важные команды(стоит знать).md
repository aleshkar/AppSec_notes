1. ping — проверка доступности хоста по ICMP, измерение времени ответа
2. tracert / traceroute — трассировка маршрута пакета до цели (промежуточные роутеры)
3. ipconfig / ifconfig / ip addr — просмотр и настройка сетевых интерфейсов (IP, маска, шлюз)
4. nslookup / dig — запросы к DNS-серверам (A, MX, TXT, NS и т.д.)
5. netstat / ss — просмотр активных соединений, открытых портов и слушающих сервисов
6. curl / wget — выполнение HTTP-запросов, скачивание файлов, проверка заголовков
7. ssh / scp — безопасное подключение к удалённому серверу и копирование файлов
8. ls, cd, grep, cat, find — навигация по файлам, поиск и фильтрация текста
9. tar, zip / unzip — создание и распаковка архивов (сжатие/разархивирование)
10. ps, top / htop, kill — просмотр процессов, мониторинг нагрузки и завершение задач
